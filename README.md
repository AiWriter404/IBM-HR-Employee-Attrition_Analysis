# HR Employee Attrition Analysis

## Project Overview

This project analyzes employee attrition patterns using the IBM HR Employee Attrition dataset.

The goal is to identify factors that contribute to employee turnover and provide actionable business insights through data analysis and visualization.

---

## Dataset Information

* Dataset: IBM HR Employee Attrition
* Records: 1,470 Employees
* Target Variable: Attrition

---

## Project Workflow

Dataset → Data Cleaning → Exploratory Data Analysis → Visualization → Business Insights

---

## Analysis Objectives

* Understand employee attrition patterns
* Identify factors affecting attrition
* Analyze demographics and job-related factors
* Generate actionable business insights
